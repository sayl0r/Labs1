## О проекте

**TaskFlow** — это программа для управления задачами, работающая в командной строке. Она создана для **разработчиков** и *системных администраторов*, которые предпочитают не покидать терминал для планирования своего дня.

Проект написан на `Python 3.12` с использованием `sqlite3` для хранения данных. Поддерживает теги, приоритеты и фильтрацию.

> *«Лучший способ предсказать будущее — создать его самому.»*  
> — Питер Друкер

Цитата отражает философию TaskFlow: не ждать, пока задачи решатся сами, а управлять ими осознанно[^1].

[^1]: TaskFlow вдохновлён методологиями GTD (Getting Things Done) и Eisenhower Matrix.

## Возможности

- 📝 **Создание, редактирование и удаление задач**
- 🏷️ **Теги** для категоризации (`работа`, `личное`, `учеба`)
- 🎯 **Приоритеты** (`high`, `medium`, `low`)
- 🔍 **Фильтрация** по статусу, приоритету, тегу
  - По статусу: `выполнено`, `в процессе`, `отложено`
  - По приоритету: `high`, `medium`, `low`
  - По тегам: любой пользовательский тег
- 💾 **Автосохранение** в базу данных SQLite
- 📊 **Статистика** выполненных и просроченных задач

### Команды CLI

| Команда | Описание |
|---------|----------|
| `list` | Показать все задачи |
| `add` | Добавить новую задачу |
| `edit <id>` | Редактировать задачу |
| `done <id>` | Отметить задачу выполненной |
| `delete <id>` | Удалить задачу |
| `filter --priority high` | Фильтрация по приоритету |
| `stats` | Показать статистику |
